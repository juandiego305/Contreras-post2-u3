## 🏗️ Laboratorio: Patrones Estructurales (Decorator & Facade)

Este proyecto modela un sistema de servicios para una tienda virtual utilizando Spring Boot. 
Se enfoca en la implementación de capas opcionales de procesamiento sobre un servicio base y el uso de una fachada unificada para simplificar subsistemas complejos.

## 🛠️ Tecnologías y Prerrequisitos

Java: Versión 17 o superior.

Framework: Spring Boot (spring-context).

Gestor de Dependencias: Maven 3.8+.

Pruebas: JUnit 5.

## 🚀 Instrucciones de Ejecución
Para verificar que el proyecto cumple con los requisitos técnicos, ejecute los siguientes comandos en su terminal:

```bash
mvn clean package
```
